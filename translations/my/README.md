## 🌱 စတင်ရန်

ဒီသင်တန်းတွင် AI Agents တည်ဆောက်ခြင်းအခြေခံကို လေ့လာနိုင်မည့် သင်ခန်းစာ ၁၁ ခု ပါဝင်သည်။ သင်ခန်းစာတစ်ခုစီသည် မိမိအကြောင်းအရာကို ဖော်ပြထားပြီး မိမိစိတ်ကြိုက်နေရာမှ စတင်နိုင်ပါသည်။

ဒီသင်တန်းတွင် Multi-language အထောက်အပံ့ ပါဝင်သည်။ [ဘာသာစကားများ](../..) ကို ဒီနေရာတွင် ကြည့်ပါ။

Generative AI မော်ဒယ်များနှင့် ပထမဆုံးအကြိမ် တည်ဆောက်မည်ဆိုပါက [Generative AI For Beginners](https://aka.ms/genai-beginners) သင်တန်းကို ကြည့်ပါ။ ဒီသင်တန်းတွင် GenAI ဖြင့် တည်ဆောက်ခြင်းအကြောင်း သင်ခန်းစာ ၂၁ ခု ပါဝင်သည်။

[ဒီ repo ကို 🌟 (star)](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) နှင့် [fork](https://github.com/microsoft/ai-agents-for-beginners/fork) လုပ်ပြီး code ကို run လုပ်ရန် မမေ့ပါနှင့်။

### လိုအပ်သောအရာများ

ဒီသင်တန်းရှိ သင်ခန်းစာတစ်ခုစီတွင် code နမူနာများ ပါဝင်ပြီး code_samples folder တွင် ရှိသည်။ [ဒီ repo ကို fork](https://github.com/microsoft/ai-agents-for-beginners/fork) လုပ်ပြီး မိမိရဲ့ copy ကို ဖန်တီးနိုင်ပါသည်။

ဒီ exercises တွင်ပါဝင်သော code နမူနာများသည် Language Models နှင့် အလုပ်လုပ်ရန် Azure AI Foundry နှင့် GitHub Model Catalogs ကို အသုံးပြုသည်-

- [Github Models](https://aka.ms/ai-agents-beginners/github-models) - အခမဲ့ / အကန့်အသတ်ရှိ
- [Azure AI Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - Azure Account လိုအပ်သည်

ဒီသင်တန်းတွင် Microsoft မှ AI Agent frameworks နှင့် services အောက်ပါအရာများကို အသုံးပြုသည်-

- [Azure AI Agent Service](https://aka.ms/ai-agents-beginners/ai-agent-service)
- [Semantic Kernel](https://aka.ms/ai-agents-beginners/semantic-kernel)
- [AutoGen](https://aka.ms/ai-agents/autogen)

ဒီသင်တန်း၏ code ကို run လုပ်ရန် အချက်အလက်များကို [Course Setup](./00-course-setup/README.md) တွင် ကြည့်ပါ။

ဤပရောဂျက်တွင် ပရောဂျက်များ၊ ထုတ်ကုန်များ သို့မဟုတ် ဝန်ဆောင်မှုများအတွက် အမှတ်တံဆိပ်များ သို့မဟုတ် လိုဂိုများ ပါဝင်နိုင်ပါသည်။ Microsoft အမှတ်တံဆိပ်များ သို့မဟုတ် လိုဂိုများကို အသုံးပြုရန် အခွင့်အရေးရှိပြီး [Microsoft's Trademark & Brand Guidelines](https://www.microsoft.com/legal/intellectualproperty/trademarks/usage/general) ကို လိုက်နာရမည်ဖြစ်သည်။
Microsoft အမှတ်တံဆိပ်များ သို့မဟုတ် လိုဂိုများကို ပြင်ဆင်ထားသော ဤပရောဂျက်တွင် အသုံးပြုခြင်းသည် Microsoft မှ ပံ့ပိုးမှုရှိသည်ဟု အလွဲသုံးစားမဖြစ်စေရန် သေချာရမည်။ တတိယအဖွဲ့အစည်း၏ အမှတ်တံဆိပ်များ သို့မဟုတ် လိုဂိုများကို အသုံးပြုခြင်းသည် အဆိုပါ အဖွဲ့အစည်းများ၏ မူဝါဒများကို လိုက်နာရမည်။

**ဝက်ဘ်ဆိုက်မှတ်ချက်**:  
ဤစာရွက်စာတမ်းကို AI ဘာသာပြန်ဝန်ဆောင်မှု [Co-op Translator](https://github.com/Azure/co-op-translator) ကို အသုံးပြု၍ ဘာသာပြန်ထားပါသည်။ ကျွန်ုပ်တို့သည် တိကျမှန်ကန်မှုအတွက် ကြိုးစားနေပါသော်လည်း၊ အလိုအလျောက်ဘာသာပြန်ဆိုမှုများတွင် အမှားများ သို့မဟုတ် မမှန်ကန်မှုများ ပါဝင်နိုင်သည်ကို သတိပြုပါ။ မူလဘာသာစကားဖြင့် ရေးသားထားသော စာရွက်စာတမ်းကို အာဏာတည်သော ရင်းမြစ်အဖြစ် သတ်မှတ်သင့်ပါသည်။ အရေးကြီးသော အချက်အလက်များအတွက် လူပညာရှင်များမှ ဘာသာပြန်ဆိုမှုကို အကြံပြုပါသည်။ ဤဘာသာပြန်ဆိုမှုကို အသုံးပြုခြင်းမှ ဖြစ်ပေါ်လာသော နားလည်မှုမှားမှုများ သို့မဟုတ် အဓိပ္ပါယ်မှားမှုများအတွက် ကျွန်ုပ်တို့သည် တာဝန်မယူပါ။
